# QR Code Generator with Styled Features

## Overview
This Python script generates a **highly customized QR code** with:
- **Rounded module edges** 
- **Gradient color styling** 
- **Embedded logo** at the center

This setup is ideal for **personal websites, portfolio links, product marketing, and brand identification**.

## Features
✅ Uses the `qrcode` library for QR code creation.  
✅ Applies styling with **RoundedModuleDrawer** for smooth edges.  
✅ Adds a **RadialGradiantColorMask** for soft gradient effects.  
✅ Overlays a **custom logo** at the center of the QR code.
